Official plugin for SpeakerText. Add SpeakerBar to your site and SEO your video transcripts.

== Description ==

SpeakerText is a service which allows you to place an interactive transcript plugin, the "SpeakerBar" below
videos on your site. Each transcript is loaded server-side, to enable Google and other search engines to index
your video content.

Once you install the plugin and enter your API key, no further action is required!  The SpeakerBar will load
*automatically* under all videos embedded on your site which have a transcript available.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `speakertext` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Add your SpeakerText public API key to the SpeakerText settings page.
